# OddOneOut-PythonProject

Odd One Out Game
Overview
The Odd One Out Game is a simple Python game where players must guess the odd number in a randomly generated 10x10 grid. The game uses Tkinter for the GUI and Pythonâ€™s random module for number generation.

How to Play
A 10x10 grid is displayed with one odd number.

Enter your guess in the input field.

Click "Submit" to see if your guess is correct.

Requirements
Python 3.x (Tkinter included)

Run the Game
bash:
python main.py
